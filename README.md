# Algorithm Challenges

- Clone this project
```
- $ git clone https://github.com/ahmetbozaci/microverse-algorithm-challenges.git`
- $ cd microverse-algorithm-challenges
```
- Install npm packages
```
- $ npm install
```
- Testing
```
- $ npm test folder-name
- Example -> $ npm test recursion
```
